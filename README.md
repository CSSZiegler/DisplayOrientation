DisplayOrientation
=======================

Application to Showcase behavior of form contents on changing Display Orientation


# Purpose
This application will showcase how the content of the form will adjust when you change the Display orientation

# Supported platforms:
**Mobile**
 * iPhone
 * Android
 * BlackBerry
 * windows phone 7.5
 * windows phone 8
 * SPA iPhone
 * SPA Android
 * SPA BlackBerry
 
**Tablet** 
 * Android
 * windows 8
 * iPad
 * SPA iPad
 * SPA Android
