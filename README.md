DeviceOrientation
=======================

Application to Showcase behavior of form contents on changing device orientation


# Purpose
This application will showcase how the content of the form will adjust when you change the device orientation

# Supported platforms:
**Mobile**
 * Android
 * iPhone
 * SPA Android
 * SPA iPhone
 
**Tablet** 
 * Android
 * iPad
 * SPA iPad
 * SPA Android
